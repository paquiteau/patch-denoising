.. py:function:: eig_analysis(input_data, max_eig_val=10)

   Return the eigen values and vectors of the autocorrelation of the patch.

   This method is surprisingly faster than the svd, but the eigen values
   are in increasing order.

   :param input_data: A 2D Array
   :type input_data: np.ndarray
   :param max_eig_val: For faster results, only the ``max_eig_val`` biggest eigenvalues are
                       computed. default = 10
   :type max_eig_val: int, optional

   :returns: * **A** (*numpy.ndarray*) -- The centered patch A = X - M
             * **d** (*numpy.ndarray*) -- The eigenvalues of A^H A
             * **W** (*numpy.ndarray*) -- The eigenvector matrix of A^H A
             * **M** (*numpy.ndarray*) -- The mean of the patch along the time axis

.. py:function:: marshenko_pastur_median(beta, eps=1e-07)

   Compute the median of the Marchenko-Pastur Distribution.

   :param beta: aspect ratio of a matrix.
   :type beta: float
   :param eps: Precision Parameter
   :type eps: float

   :returns: **float**
   :rtype: the estimated median

   .. rubric:: Notes

   This method Solve :math:`F(x) = 1/2` by dichotomy with
   .. math ::

       F(x) = \int_{\beta_{-}}^{x} \frac{\sqrt{(\beta_{+}-t)(t-\beta_{-})}}{2\pi\beta t} \mathrm{d}t

   The integral is computed using scipy.integrate.quad
