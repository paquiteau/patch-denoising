denoiser.simulation.noise.add_temporal_gaussian_noise(array, sigma=1, g_factor_map=None, rng=None)[source]#

Add gaussian noise to array.

Parameters:

  • array (numpy.ndarray) – The noise_free ND-array, where the last dimension is a dynamical one (e.g. time)

  • sigma (float) – gaussian noise variance

  • g_factor_map (numpy.ndarray, optional) – Spatial variation of the noise ((N-1)D array). default is identity.

Returns:

A noisy array

Return type:

numpy.ndarray

denoiser.simulation.noise.add_temporal_rician_noise(array, scale=1, rng=None)[source]#

Add center rician noise to array.

Parameters:

  • array (numpy.ndarray) – The noise-free array

  • sigma (float) – The scale of the Rice distribution

Notes

This function considered centered Rician noise [1], and thus the noise generated follows a Rayleigh distribution [2].
